01The agent is the bridge

No automatic link wires code-facts to decisions. The agent connects them — queries graphify for structure, reads memory for intent. The reasoning loop is the bridge.

02Sprawl is the real risk

Five surfaces can become five places to lose things. One lane each — and never hand-curate the second brain. It's generated; manual edits get overwritten.

The payoff isn't a smarter model. It's a system that never forgets what we already learned.
