Two Honest Truths

where the leverage stops
01The agent is the bridge

There is no automatic link between code-facts and decisions. The agent connects them — it queries graphify over MCP for structure and reads memory for intent. Nothing wires the two together but the reasoning in the loop.

02Sprawl is the real risk

Five surfaces can become five places to lose things. The discipline holds it together: one lane each, and never hand-curate the second brain — it's generated, so manual edits get overwritten on the next refresh.
